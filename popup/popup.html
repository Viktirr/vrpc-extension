About

VRPC Extension

Version:

App and extension version mismatch. Please update both to the latest version.

General

VPRC at its core was designed to be a set of software that allows communication between the browser and Discord, plus some other nice-to-have features.

The application can also store data of what you've listened to, for how long, which platform you've listened on, etc. You're free to opt-out of this feature by disabling "Enable Listening Data" on Configuration if available. We currently don't upload anywhere and is ONLY stored locally, however uploading is a planned feature.

Miscellaneous & Disclaimers

You may access your listening activity if you have Listening Data enabled in:

%appdata%\VRPCApp\ListeningData.json

Some settings don't always apply immediately. At most you'll have to reload the website that is currently recognised by the extension.

Some icons are used from feathericons.com

VRPC Extension is not affiliated with Discord, YouTube Music or Soundcloud.

Troubleshooting

If the extension can't connect to the app, it's either because the app is not recognised, not installed or not running. All of these should work automatically, however, if it doesn't work, please visit the Troubleshooting section over at

https://github.com/Viktirr/vrpc-app

Currently, installation is manual, and installation instructions are at the Installation portion of the GitHub above.
