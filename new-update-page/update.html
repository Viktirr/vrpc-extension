VRPC Update/Installation Information!

VRPC has just been updated (or this might be the first time you've installed it)! The changes between this version and the previous one are drastically different and the program will break if not updated correctly.



Installation & Update Guide

Hereâ€™s what you should do if updating:

Uninstalling (skip if you're installing the application for the first time)

  1. Open Settings/Control Panel
  2. Head to Apps
  3. Uninstall VRPC Application
  4. When prompted press Y and press Enter to uninstall

Installing

  1. Download the newest version available here
  2. Extract the downloaded `vrpc-full.zip` package inside a folder.
  3. Open (Run) `VRPC.exe` from the extracted folder.
  4. Type Y and press Enter to install
  5. Restart the browser

Please follow these steps exactly as written as doing otherwise will break the application entirely (there are changes in the application that could be fatal if done in another way).

If in fact, something breaks, delete the %appdata%\VRPCApp folder. Feel free to keep/copy `ListeningData.json` available inside the Data folder of VRPCApp.

For more general information head over to: https://viktir.com/vrpc

How do I know everything's working?

When selecting the popup tab of the extension, available on most browsers under the puzzle icon on the browser toolbar. The top-most text should say "Connected" or "Rich Presence inactive" depending on where you're looking, sometimes there's a small delay when launching for the first time. If not check out the troubleshooting steps

How to use the application?

The application and all its settings can be viewed and modified in the popup tab of the extension, to access it, press the extension icon on your browser toolbar (usually a puzzle icon) and press the VRPC Extension.

Clicking on Stats, Configuration and About will bring different menus. You may also see your Rich Presence status at the top of the extension popup.

Here's what's changed between v0.81 and v1.00:

Extension Changes

  • Configuration categories - No longer an eyesore to look for a specific settings
  • Soundcloud works when signed out - You no longer have to log in to have Soundcloud as your Rich Presence
  • Add song duration to Stats - You can now view the song duration in stats (this could be useful for later)
  • Hide Stats when no data is present - Stats can be disorienting when there's no data being present
  • Fix "and undefined" - "and undefined" used to say when there was no data on some Stats, especially when creating data for a song that was not registered before. This has now been replaced with showing "0 seconds" instead.
  • App Locked Warning - If there are multiple browsers running and this one isn't used for VRPC, there will be a warning saying that the application will not run because it is locked by another browser.
  • YouTube Music rework - Reworked how some data was gathered on YouTube Music, things should be a little bit more stable now.
  • New font: Roboto
  • Fix links in Soundcloud

Application Changes

  • Configuration categories
  • Installation files are now in bin - Less cluttered when browsing the installation folder of VRPC
  • Lock file - Due to conflicts with having multiple browsers open with VRPC, especially with Rich Presence, I now only allow one instance of the application running, this resolves many issues that come with having multiple instances of the application open (such as conflicts when saving with Listening Data or Rich Presence overlap)
  • Song Duration in Listening Data
  • Listening Data Backup - If the newest version of Listening Data failed to save, the application will use a backup instead to ensure no major data is lost
  • YouTube Music Rework - Some functionality of YouTube Music has been reworked to fix some issues, such as pausing not working right or the timestamps being off. With this also came the removal of the forced 2 second delay when switching songs as now it works without it
  • Listening Data Heartbeat fix - Sometimes Listening Data will wrongly assume that the song is not currently being played. This has been fixed.
  • Creation Date on Listening Data - The date and time of the Listening Data file being created
  • Remove infinite Rich Presence - If Rich Presence lasts longer for the length of a song, it will be automatically removed
  • Clamp to 128 bytes instead of arbitrary 56 characters - Allows for longer lengths of song names being used, this is to combat the previous 56 character limit I arbitrarily put. Allows you to have super lengthy song names correctly displayed now.

Thank you for using VRPC!
